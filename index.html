Key Features

Large “Time Remaining” Countdown

Displays a clear countdown message (Time remaining: HH:MM:SS) designed for intraday, time-based bars.

Yellow + Flashing Red Thresholds

Turn the text yellow under YellowUnderSeconds and flash red under FlashRedUnderSeconds with configurable speed (FlashToggleMs).

Drag & Drop Placement (Optional)

Enable EnableDragDrop to move the countdown anywhere on the chart and persist its position using internal DragX/DragY.

Sound Alerts (Yellow + Red)

Play optional sounds when thresholds are crossed: EnableYellowSound / EnableRedSound, each with its own selected file and volume.

Multi-Sound Modes

Use MultiSoundMode per channel: First, Random, or RoundRobin using a single file plus an optional list.

Smart Sound File Dropdown (NT8.1-safe)

Dropdown merges and auto-refreshes sound files from: Program Files, Documents, and an optional Custom folder—no restart required.

How It Works

1

Verify Chart Type + Connection

Works on intraday time-based bars. Displays helpful status messages if disconnected, outside session hours, or waiting on real-time data.

2

Update Countdown via UI Timer

A dispatcher timer updates the “Time remaining” value frequently (based on your flash toggle), keeping the display responsive without needing a full bar update.

3

Apply Color + Flash Rules

Above yellow: base text color. Under yellow: yellow. Under red: flashing red using FlashToggleMs.

4

Trigger Sound Alerts (Optional)

Sounds can fire once when thresholds are crossed and optionally repeat during the red flash window using RepeatRedSoundDuringFlash and RepeatRedIntervalMs.

Frequently Asked Questions

What charts does MQT_BarTimer work on?

It’s intended for intraday time-based bars. If you apply it to non-time-based bars, it will show a message indicating the requirement.

Can I move the countdown text?

Yes. Enable EnableDragDrop, then click and drag the text on the chart. The position persists using DragX/DragY.

How do I change when it turns yellow or flashes red?

Use YellowUnderSeconds and FlashRedUnderSeconds. Control flash speed with FlashToggleMs.

How do sound alerts work?

Sounds can trigger once when the countdown crosses below the yellow/red thresholds. You can also repeat the red sound during the flash window with RepeatRedSoundDuringFlash.

Where does the sound dropdown pull files from?

The dropdown merges Program Files sounds, Documents sounds, and an optional Custom folder if enabled. It rescans when you open the dropdown (no restart needed).

Can I rotate between multiple sounds?

Yes. Add extra files to YellowSoundList or RedSoundList and set YellowSoundMode / RedSoundMode to Random or RoundRobin.
